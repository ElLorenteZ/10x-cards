### Main problem 
Manual creation of high quality flashcards for learning is time consuming, that discourage from using spaced repetition for effective learning.

### Minimal set of functionalities
- Using AI to flashcard generation based on introduced text (copy-paste),
- Manual flashcard creation,
- Preview, edition and deletion of flashcards,
- Simple account system for flashcards storage,
- Integration with ready spaced repetition algoritm.

### What is not part of MVP
- Self, advanced repertition algorithm (like SuperMemo, Anki)
- Import of multiple data formats,
- Sharing flashcards set between users,
- Integration with other education platforms,
- Mobile apps (web first),
- Caregories of flashcards sets.

### Success criteria
- 75% flashcards generated by AI is accetped by user,
- 75% of flashcards is generated with using AI.
